# 📖 Reading Diary (Щоденник читання)

## 📌 Опис
Це проста програма на Python, яка працює як щоденник читання.  
Користувач вводить кількість прочитаних сторінок за кожен день тижня, після чого програма обчислює результат.

---

## ⚙️ Можливості
- Введення даних за 7 днів
- Перевірка правильності введення
- Підрахунок:
  - загальної кількості сторінок
  - середнього значення
- Оцінка результату:
  - менше 5 → *Потрібно читати більше*
  - 5–15 → *Хороший результат*
  - більше 15 → *Відмінна робота!*

---

## 🚀 Запуск програми
1. Встановіть Python  
2. Завантажте файл `diary.py`  
3. Запустіть у терміналі:
   ```bash
   python diary.py
